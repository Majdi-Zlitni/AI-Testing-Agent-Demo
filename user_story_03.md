Test Case: View a Product's Detail Page
Goal: A user should be able to click on a product to view its dedicated detail page and then return to the main inventory.

Test Steps:

Log in to https://www.saucedemo.com with the username standard_user.
On the inventory page, click on the name of the product "Sauce Labs Onesie".
Verify you are on the correct product detail page by checking that the URL now includes inventory-item.html?id=5.
Verify the text "Sauce Labs Onesie" is visible on the page.
Click the button with the ID back-to-products.
Verify you have returned to the main inventory page (/inventory.html).
